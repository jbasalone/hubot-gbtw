## Installation

Add **hubot-gbtw** to your `package.json` file:

```json
"dependencies": {
  "hubot": ">= 2.5.1",
  "hubot-scripts": ">= 2.4.2",
  "hubot-gbtw": ">= 0.0.0"
}
```

Add **hubot-gbtw** to your `external-scripts.json`:

```json
["hubot-gbtw"]
```

Run `npm install hubot-gbtw`
