## Installation

Add **hubot-fail** to your `package.json` file:

```json
"dependencies": {
  "hubot": ">= 2.5.1",
  "hubot-scripts": ">= 2.4.2",
  "hubot-fail": ">= 0.0.0"
}
```

Add **hubot-fail** to your `external-scripts.json`:

```json
["hubot-fail"]
```

Run `npm install hubot-fail`
